XSS + Cookie Stealing — Curi Token Rahasia

Sebuah portal nilai menyimpan informasi rahasia dalam cookie browser. Cookie ini tidak memiliki atribut HttpOnly, artinya JavaScript bisa membacanya. Gunakan XSS untuk mencuri cookie dan temukan flag!

💡 Tampilkan Hint

Cookie bernama secret_flag berisi flag. Gunakan payload:
<img src=x onerror=alert(document.cookie)>
Buka halaman rentan dulu agar cookie di-set oleh server.

🔗 Buka Portal Nilai 📚 Tutorial

🎯 Cara Mengerjakan

1. Buka "Portal Nilai" (halaman rentan)

2. Server akan set cookie secret_flag yang berisi flag

3. Gunakan XSS di form pencarian untuk membaca cookie tersebut

4. Alert akan menampilkan semua cookie — cari nilai secret_flag

// Payload untuk membaca cookie:
<img src=x onerror=alert(document.cookie)>

// Output yang kamu cari:
secret_flag=CTF{...}
⚠️ Ingat: Cookie HttpOnly TIDAK bisa dibaca JS. Di sini cookie sengaja dibuat tanpa HttpOnly!
