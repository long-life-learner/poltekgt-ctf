Stored XSS — Buku Tamu Berbahaya

Sistem buku tamu menyimpan komentar ke database dan menampilkannya tanpa filter. Injeksikan payload XSS yang tersimpan — setiap orang yang membuka halaman akan menjalankan kode kamu!

🔗 Buka Buku Tamu 📚 Tutorial

🎯 Cara Mengerjakan

1. Buka halaman Buku Tamu di tab baru

2. Isi kolom Komentar dengan payload XSS

3. Klik "Kirim Komentar" — payload akan tersimpan di database

4. Halaman akan reload dan payload kamu akan dieksekusi (karena tidak ada sanitasi!)

5. Alert box akan menampilkan flag tersembunyi

// Payload untuk Stored XSS:
<img src=x onerror=alert(document.getElementById('flag-stored').innerText)>
